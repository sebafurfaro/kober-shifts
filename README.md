# Kober Shifts

Gestor de turnos y agenda para profesionales, negocios y centros de salud.

## ¿Por qué elegir Kober Shifts?

- Reduce ausencias y olvidos con recordatorios automáticos por WhatsApp, email y SMS.
- Agenda inteligente: visualizá y gestioná turnos desde cualquier dispositivo.
- Panel de administración para controlar roles, permisos y coberturas.
- Integración con Google Calendar para profesionales.
- Personalización de branding y comunicación para tu negocio.
- Seguridad y privacidad: cada usuario accede solo a lo que necesita.

## Beneficios para tu negocio

- Más clientes presentes, menos tiempo perdido.
- Experiencia profesional y moderna para tus pacientes o clientes.
- Automatización de tareas administrativas.
- Panel multirol: administradores, profesionales, recepcionistas y pacientes.
- Coberturas, servicios y sedes configurables.
- Notificaciones personalizadas.

## ¿Cómo funciona?

1. Registrá tu negocio y configurá tu panel.
2. Agregá profesionales, servicios y sedes.
3. Tus clientes pueden reservar turnos online.
4. Reciben recordatorios automáticos antes de cada cita.
5. Gestioná todo desde un panel centralizado.
